# Film Folio
영화리뷰를 편리하게 볼 수 있는 플랫폼

## 팀원 소개 
 - 팀원 : 이주원 김다은 이채영 김지은

## 개발 일정 
 - 08.02 ~ 08.04

## 기획의도 
 - 현재 개봉한 영화만을 대상으로 리뷰 조회 기능
 - 장르별 / 평점별 리뷰 조회 기능
 - 내가 작성한 리뷰글 조회 기능
 - 내 리뷰 랭킹을 조회

## 개발 환경 
  - 언어 : Java 11 , OracleDataBase 21C XE
  - 개발 Tool : eclipse IDE , SQL DEVELOPER
  - DB : Oracle
  - 협업 Tool : Github, Matter Most, Kakao Talk
